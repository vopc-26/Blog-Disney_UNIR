### Proyecto Blog de Disney favoritos V. P.

Este proyecto es un blog de mis gustos por Disney, es una sitio sencillo que permite a los usuarios iniciar sesión 
y explorar mediante ventanas la películas, música que me gustan y una traivia divertida.

## Estructura de Archivos

- `login.html`: Página de inicio de sesión.
- `index.html`: Página principal con enlaces a ventanas de favoritos.
- `peliculas.html`, `musica.html`, `trivia.html`: Páginas individuales para cada sección.
- `assets/`: Carpeta que contiene los recursos del proyecto.
  - `css/`: Contiene el archivo `style.css` para el estilo del blog.
  - `js/`: Contiene el archivo `login.js` para la funcionalidad de inicio de sesión.
  - `img/`: Carpeta que contiene las imágenes utilizadas en el blog.
  - `audio/`: Carpeta que contiene los audios utilizadas en el blog

## Instalación

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.
